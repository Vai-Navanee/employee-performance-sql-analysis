# 📊 ScienceQtech Employee Performance Mapping (SQL Project)

## 📌 Overview
This project analyzes employee performance, salary structure, and project allocation for ScienceQtech, a Data Science startup. The goal is to support HR decision-making using SQL-based analytics. This project simulates real-world HR analytics to help organizations optimize workforce performance and salary distribution using SQL.

---

## 🎯 Objective
To evaluate employee performance, salary distribution, experience levels, and job role alignment using SQL queries and database operations.

---

## 🧰 Tools & Technologies
- MySQL
- SQL (Joins, Subqueries, Window Functions)
- Stored Procedures & Functions
- Database Design

---

## 📁 Dataset
- emp_record_table
- proj_table
- data_science_team

---

## 🧩 Key SQL Operations Performed

### ✔ Basic Analysis
- Employee listing by department
- Rating-based filtering
- Salary bonus calculation
- Department-wise data extraction

### ✔ Advanced Analysis
- Employee reporting hierarchy
- Department-wise performance comparison
- Salary distribution by role
- Experience-based ranking

### ✔ Database Features Used
- Views (High salary employees)
- Stored Procedures (Experience-based filtering)
- Functions (Job role classification)
- Indexing (Query optimization)

---

## 📊 Business Insights
- Employee experience strongly influences salary levels
- Performance ratings impact bonus allocation
- Structured job roles improve workforce clarity
- Database optimization improves query performance

---

## 🧠 Outcome
This project demonstrates SQL-based HR analytics, covering querying, database design, and performance optimization for real-world business decision-making.
